AgentProof benchmark

The adversarial benchmark for AI agent gate integrity.

AgentProof is a standalone, language-agnostic benchmark for autonomous coding agents. Each scenario runs a real adversarial attempt against a repository's controls and confirms it is caught, so a passing score is verifiable evidence rather than a claim. Any agentic system can run it, which makes it a practical conformance test for the field.

$ node agentproof/runner.mjs → Score: 25/25 normative | 10/10 extended (35/35 total) · HARDENED
HARDENED = 25/25 normativePARTIAL = 20 to 24UNHARDENED = 19 or fewer

Why gate integrity matters

An agent that can weaken a test, skip it, cast a type to any, or lower a coverage floor can turn a red pipeline green without fixing anything. Gate integrity is the property that these moves are caught by controls the agent cannot edit. AgentProof measures it directly: it attempts each attack and verifies the gate holds. Modonome scores 25/25 HARDENED, and every one of the 25 normative scenarios is machine-verified.

Run it yourself

AgentProof lives in the repository and runs with Node. Clone the repo and run the scenarios against the controls.

# clone and run the full benchmark
git clone https://github.com/enumind/modonome
cd modonome
node agentproof/runner.mjs
→ Score: 25/25 normative | 10/10 extended (35/35 total)  Level: HARDENED

The full scenario descriptions, expected outcomes, and source are in the AgentProof documentation. See how the loop uses these controls on the Modonome home page or read the quickstart.
